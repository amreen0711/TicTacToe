🎮 Tic Tac Toe Android App
📌 Overview

This project is a fully functional Tic Tac Toe mobile application developed using Java and Android Studio. The app allows two players to compete in a classic turn-based game with a smooth user interface, real-time feedback, and persistent score tracking.

Features
Turn-based gameplay (Player X vs Player O)
Win, lose, and draw detection logic
Score tracking across multiple rounds
Game reset functionality
Interactive and responsive UI
Visual feedback for player actions (highlighting, animations)
Persistent data storage using SharedPreferences
Technologies Used
Java (Core game logic)
Android Studio (Development environment)
XML (UI design and layouts)
SharedPreferences (Local data persistence)
Core Concepts Implemented
2D Array (Game Board Matrix):
Used to manage and track the state of the game board efficiently.
Event-Driven Programming:
Handled user interactions using button listeners for real-time gameplay updates.
State Management:
Controlled player turns, round counts, and game outcomes systematically.
Persistent Storage:
Used SharedPreferences to store player scores across app sessions.
